The Pilapse Project

I wanted to document the building of this car with a time-lapse video but couldnt find any programs out there that I found adequate. Thus, I decided to build my own using a Raspberry Pi Zero. The major features provided by pilapse are as follows:

  • Still image capture and simulatenous video encoding
  • Recording session labeling and timeline support
  • Backup to a secondary server
  • Video access and capture control via internally served web page.
  • User authentication and new video notification
Pilapse is written in python and utilizes ffmpeg, sqlite, flask, jinja, and peewee. The source code is available on github as Pilapse and is available for use, free of charge.
